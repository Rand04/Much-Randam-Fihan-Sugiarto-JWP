## Deskripsi
Aplikasi ini digunakan untuk mengelola data produk cat serta memberikan rekomendasi berdasarkan penjualan, harga, dan kualitas.

---

## Fitur
- Login & Autentikasi (Email: admin@gmail.com, Password: admin123)
- CRUD Produk Cat
- Dashboard (statistik & grafik)
- Rekomendasi Produk
- Filter Produk

---

## Database
Tabel yang digunakan:
- users (login)
- paints (produk cat)

---

## Teknologi
- Laravel
- MySQL
- Tailwind CSS
- Chart.js

---

## Cara Menjalankan
1. Clone project
2. Jalankan:
   composer install
3. Copy .env
4. Setup database
5. Jalankan:
   php artisan serve
